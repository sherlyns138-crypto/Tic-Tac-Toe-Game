# Tic-Tac-Toe-Game
A quick fun game to play.

- I created this Tic Tac Toe game using html, css and javascript 
- This small game can be a timepass to relax the mind

# Changes in index.html
- It is the skeleton of the game
- Did not do much work in this file, but it is lot more important as it acts as the framework for the entire game

# Changes in style.css
- It is used to add designs and creative work for the webpage
- Website looks more beautiful and colourful due to this css file 
- It is used to build the grids for the game and also buttons hovering and color changing 

# Changes in script.js
- It is the main file which plays a major role in adding interactivity to the webpage
- Adding different ideas and making the page look enjoyable


All together makes a beautiful webpage which is neat, attractive and interactive.
